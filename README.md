# Новый сайт «Автобан-Юг»

Статический сайт без сборки и зависимостей. Главная страница открывается из файла `index.html`, каталог наполняется из `data/catalog.js`.

## Что внутри

- `index.html` - новая витрина: sticky-навигация, слайды, каталог, заявка в WhatsApp, карта, контакты.
- `styles.css` - дизайн, адаптив, переходы, sticky-блоки и анимации появления.
- `scripts.js` - слайдер, фильтры каталога, поиск, заявка и модальные карточки.
- `data/catalog.js` - вся информация для каталога, контактов, фото-шаблонов и слайдов.
- `data/catalog-full.js` - расширенный слой каталога: разделы старого сайта и марки авто с логотипами.
- `admin.html` - простой редактор карточек каталога с экспортом обновленного JS.
- `assets/` - места для реальных фотографий магазина и товаров.

## Куда загружать фото

Фирменные материалы уже перенесены со старого сайта:

- `assets/img/brand/LogoAB.jpg` - старый логотип «Автобан».

Главный экран:

- `assets/img/hero/warehouse-parts.jpg` - склад, полки, наличие товара.
- `assets/img/hero/store-parts.jpg` - магазин, витрина, автотовары.
- `assets/img/hero/service-parts.jpg` - подбор деталей, сервисная зона, менеджер.

Товары:

- `assets/catalog/brakes/brake-kit.webp` - тормозные диски, колодки.
- `assets/catalog/oils/motor-oil.webp` - масла и жидкости.
- `assets/catalog/service/filters.webp` - фильтры и расходники.
- `assets/catalog/suspension/suspension.webp` - подвеска.
- `assets/catalog/electric/battery.webp` - аккумуляторы и электрика.
- `assets/catalog/lights/headlights.webp` - автооптика.
- `assets/catalog/engine/engine-parts.webp` - двигатель, ремни, свечи.
- `assets/catalog/vin/vin-order.webp` - подбор по VIN / универсальная карточка.

Пока этих файлов нет, сайт показывает временные изображения из внешних ссылок. Когда загрузите реальные фото с такими именами, они автоматически заменят временные.

## Как добавить товар

Вариант 1: откройте `admin.html`, заполните карточку, нажмите «Добавить карточку», затем экспортируйте JS и перенесите данные в `data/catalog.js`.

Вариант 2: вручную добавьте объект в массив `products` в `data/catalog.js`.

Мини-шаблон карточки:

```js
{
  id: "unique-product-id",
  title: "Название товара",
  category: "Категория",
  description: "Короткое описание для карточки",
  price: "от 1 900 ₽",
  stock: "В наличии",
  delivery: "самовывоз сегодня",
  tags: ["тег 1", "тег 2"],
  brands: ["BOSCH", "TRW"],
  priority: 10,
  image: "assets/catalog/category/photo.webp",
  fallback: "https://images.unsplash.com/..."
}
```

## Что желательно заменить перед публикацией

- Временные фото на реальные фото магазина, склада, сотрудников и товаров.
- Цены в `data/catalog.js`, если нужно показывать точные суммы.
- Координаты карты в `index.html`, если нужна точная метка входа.
- Отзывы в `data/catalog.js`, если хотите поставить реальные короткие цитаты клиентов.
