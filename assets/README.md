# Фото для сайта

Сюда кладутся реальные изображения. Рекомендуемый формат: `.webp`, ширина 1200-1800 px, вес до 400 КБ на фото.

Структура уже ожидается в `data/catalog.js`:

- `img/hero/` - фотографии для главного слайдера.
- `img/team/` - сотрудники, стойка выдачи, консультации.
- `catalog/` - фотографии товаров и категорий.

Если название файла совпадает с путем в `data/catalog.js`, сайт подхватит его автоматически.
