# Фото каталога

Рекомендуемые папки:

- `brakes/` - тормозная система.
- `oils/` - масла и автохимия.
- `service/` - фильтры и расходники.
- `suspension/` - подвеска и ходовая.
- `electric/` - аккумуляторы и электрика.
- `lights/` - автооптика.
- `engine/` - двигатель.
- `vin/` - подбор по VIN или универсальные изображения.

Путь к фото указывается в поле `image` каждой карточки в `data/catalog.js`.
